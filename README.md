# To-Do App

A simple and functional To-Do List app built with React.

## Features
- Add, edit, delete tasks
- Mark tasks as complete or important
- Search tasks by name
- Filter by All / Pending / Completed / Important
- Clear all completed tasks
- Task statistics (Total, Pending, Important)

## Tech Stack
- React.js
- CSS
- React Icons

## How to Run Locally

1. Clone the repo:
   git clone https://github.com/PalakSa08/To-Do-App.git

2. Go into the folder:
   cd To-Do-App

3. Install dependencies:
   npm install

4. Start the app:
   npm start

5. Open your browser at:
   http://localhost:3000
